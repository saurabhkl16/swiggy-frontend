# Swiggy Clone App

**simple food delivery application** where users can **sign up, log in, browse restaurants, view menus, and place orders**.  
This project is built with **Angular 21**, **Java 21 (Spring Boot)** for backend, and **MySQL** as the database.  

---

## **Tech Stack**

- **Frontend:** Angular 21, SCSS, Angular Material.
- **Backend:** Java 21, Spring Boot 3.x
- **Database:** MySQL 8.x
- **Server/Runtime:** Node.js 20+ (for Angular development server)
- **Build Tools:** Maven (backend), npm (frontend)

---

## **Features**

- **User Authentication:**
  - Sign Up / Sign In functionality
  - JWT-based authentication (optional)
  
- **Restaurant Listing:**
  - Browse available restaurants
  - View restaurant details and menu items

- **Food Ordering:**
  - Add items to cart
  - Place an order

- **Responsive UI:**
  - Works on desktop and mobile devices
